# Task Manager Frontend

React + TypeScript + Vite frontend for the Task Management assessment.

## Requirements

- Bun
- Node.js (only for tooling that Bun may delegate to)

## Environment

Create a `.env` file in this folder:

```env
VITE_API_BASE_URL=http://localhost:8000
```

## Install

```bash
bun install
```

## Run (Dev)

```bash
bun run dev
```

## Build

```bash
bun run build
```

## Preview

```bash
bun run preview
```
