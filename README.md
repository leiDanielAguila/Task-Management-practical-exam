# Task Management Application

## Overview

This project is a Fullstack Task Management Application designed to help users create, organize, and manage their daily tasks efficiently.

The application provides core task management functionality including task creation, editing, deletion, completion tracking, searching, and filtering. It follows a modern client-server architecture where the frontend communicates with a RESTful backend API to perform task operations and persist data.

The project was developed as part of a Fullstack Development assessment to demonstrate proficiency in frontend development, backend development, API design, database integration, and application architecture.

## Running the Application

### Prerequisites

- Make sure the following are installed:

- Node.js (v18 or later recommended)
- Python 3.11+ (or your project's required version)
- npm



## Features

* Create new tasks
* Edit existing tasks
* Delete tasks
* Mark tasks as complete or incomplete
* Search tasks by title
* Filter tasks by status

  * All Tasks
  * Active Tasks
  * Completed Tasks
* Combine search and filtering functionality

## Architecture

The application follows a layered architecture that separates responsibilities into distinct components:

* Frontend Layer – User interface and client-side interactions
* API Layer – HTTP endpoints and request handling
* Service Layer – Business logic and validation
* Repository Layer – Data access and persistence
* Database Layer – SQLite data storage

This separation promotes maintainability, scalability, and clean code practices.
